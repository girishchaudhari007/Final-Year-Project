# 🛍️ Fashion Rental Website

A simple Fashion Rental Website built using HTML, CSS, and JavaScript.

This project includes login, registration, product pages, and a basic front-end structure for an online fashion rental system.

---

## 🚀 Features

- 🔐 User Login Page
- 📝 User Registration Page
- 👕 Men Item Page
- 👗 Women Item Page
- 👟 Shoe Item Page
- 🛒 Shop Page
- 📩 Contact Us Page
- 🎨 Responsive UI using CSS
- ⚡ Basic Form Validation using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
